# Data Analyzer Skill

## 説明

このスキルは、データファイル（JSON、CSV、テキスト）を分析し、以下の情報を提供します：

- レコード数・行数のカウント
- 数値フィールドの統計（最大、最小、平均、合計）
- データ構造の分析
- フォーマット検証

## いつ使用するか

- ユーザーがデータファイルの分析を要求したとき
- 「データを集計して」「統計を出して」といった要求があったとき
- CSVやJSONファイルの内容を理解したいとき

## 手順

1. まず対象ファイルを Read ツールで読み込む
2. ファイル形式（JSON/CSV/テキスト）を判定
3. データ構造を解析
4. 統計情報を計算
5. 結果をわかりやすく報告

## 対応フォーマット

- **JSON**: オブジェクト配列、ネストされた構造
- **CSV**: ヘッダー付き、カンマ区切り
- **テキスト**: 行数カウント、パターン検出

## 出力例

```
## データ分析結果

ファイル: sales_data.json
形式: JSON配列
レコード数: 150

### 数値フィールドの統計
| フィールド | 最小 | 最大 | 平均 | 合計 |
|-----------|------|------|------|------|
| price     | 100  | 9999 | 2450 | 367500 |
| quantity  | 1    | 50   | 12   | 1800 |

### データ構造
- id: number
- name: string
- price: number
- quantity: number
- date: string (ISO形式)
```
